# 政数表格智能体平台

## 项目简介

政数表格智能体平台是一个面向政务、企业申报、内部审批等场景的智能表格处理系统。项目目标是将传统依赖人工完成的“复制、粘贴、查询、判断、填写”流程，升级为由 AI 驱动的自动化表格处理流程。

用户只需要上传符合规范的 Excel 表格，系统会自动读取表格中的 A、B 列内容，调用大语言模型进行分析、分类、提取或生成，并将处理结果写入 C 列，最终导出完整的 Excel 结果文件。

本项目适用于政务数据专员、窗口工作人员、政策研究员等需要批量处理表格数据的用户群体。

---

## 核心功能

### 1. Excel 表格上传

- 支持用户上传待处理的 Excel 表格
- 支持基础文件格式校验
- 支持按照预设规范读取表格内容
- 保留原始表格数据结构

### 2. 表格自动解析

系统会自动读取表格中的数据内容，并将每一行拆分为独立的处理任务。

典型输入结构如下：

| A列 | B列 | C列 |
|---|---|---|
| 原始文本内容 | 处理指令 | AI生成结果 |

例如：

| A列 | B列 | C列 |
|---|---|---|
| 缺少有效身份证明文件 | 判断不予受理原因类型 | 材料缺失 |

### 3. AI 智能处理

系统基于大语言模型完成表格内容的智能分析，支持：

- 文本分类
- 信息提取
- 内容生成
- 事实查询
- 情感分析
- 规则判断
- 批量数据处理

### 4. 批量任务拆分

为了解决大表格处理时的上下文窗口限制，系统会将完整表格拆分为多个行级任务，分别处理后再统一聚合结果。

该方式可以提升系统在大规模表格场景下的稳定性和处理能力。

### 5. 结果导出

处理完成后，系统会生成新的 Excel 文件，并保留原始 A、B 列内容，同时将 AI 生成结果写入 C 列。

用户可以一键下载最终结果文件。

## 项目价值

本项目主要解决以下问题：

1. **提升效率**

   将人工逐行判断、复制、填写的流程自动化，减少重复劳动。

2. **降低错误率**

   减少人工疲劳导致的录入错误和主观判断偏差。

3. **提高数据标准化程度**

   通过统一的 AI 处理规则，使输出结果更加规范。

4. **支持大规模处理**

   通过任务切片和异步处理机制，支持上千行甚至上万行表格的稳定处理。

5. **释放人力资源**

   让工作人员从低价值重复劳动中解放出来，专注于分析、决策和业务优化。

## 技术架构

系统整体采用分层架构设计，主要包括：

```text
用户上传 Excel
        ↓
输入层：文件接入与格式校验
        ↓
解析层：Excel 转 Markdown / JSON
        ↓
任务层：行级任务拆分与队列调度
        ↓
模型层：大语言模型推理处理
        ↓
数据层：结果清洗、聚合与写入
        ↓
导出层：生成最终 Excel 文件
